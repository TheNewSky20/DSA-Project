# 🚀 Chicken Invaders Cute Edition

A 2D arcade shooting game inspired by the classic **Chicken Invaders** series, developed using **Java** and **Java Swing** while applying **Data Structures and Algorithms (DSA)** concepts in a real-time gaming environment.

---

## 📌 Project Overview

**Chicken Invaders Cute Edition** is a 2D arcade shooting game where players control a spaceship to defeat enemy chickens, collect upgrades, and fight powerful bosses.

This project was developed as part of the **Data Structures and Algorithms (DSA)** course to demonstrate the practical application of DSA concepts in software and game development.

The game integrates several important **data structures** and **algorithms** to manage real-time gameplay, enemy behavior, collision detection, and score ranking systems.

---

## 🎮 Features

### 🐔 Enemy System
The game includes **3 different enemy types**:

- **Baby Chicken** – basic enemy with low health
- **Fast Chicken** – high movement speed
- **Tank Chicken** – high durability and stronger defense

### 👾 Boss System
Bosses appear every **5 levels** with increasing difficulty.

Implemented bosses:

- **Giant Chicken**
- **Angry Chicken**
- **Mecha Chicken**

### 🔫 Weapon & Item System

Players can collect dropped items to improve combat abilities:

- Double Shot
- Triple Shot
- Laser
- Missile
- Shield
- Heal

### 🏆 Leaderboard System

The game includes a **score ranking system**:

- Player score saving
- Top 10 leaderboard
- Score history tracking
- Automatic sorting in descending order

### ⚡ Gameplay Features

- Auto shooting system
- Infinite level progression
- Real-time collision detection
- Progressive difficulty scaling
- Dynamic enemy spawning

## 🧠 Data Structures & Algorithms Used

### Data Structures

| Data Structure | Purpose |
|---|---|
| **Array** | Enemy formation management |
| **ArrayList** | Dynamic object management |
| **Queue** | Enemy spawning system |
| **Stack** | Game state management |
| **HashMap** | Asset management |

### Algorithms

| Algorithm | Purpose |
|---|---|
| **Collision Detection** | Bullet and enemy interaction |
| **Breadth-First Search (BFS)** | Boss movement behavior |
| **Recursive Enemy Generation** | Enemy wave generation |
| **Sorting Algorithm** | Leaderboard ranking |
