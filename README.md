# Chicken Invaders Cute Edition

## Run
Open this folder in VS Code, then run `src/chickeninvaders/Main.java`.

## Controls
- Enter: Start / Restart
- W A S D or Arrow Keys: Move
- P: Pause
- Auto shooting is enabled

## Features
- Cute player and chickens
- 3 chicken types: Baby, Fast, Tank
- 3 boss types rotating every 5 levels: Giant, Angry, Mecha
- Equipment drops: Double Shot, Triple Shot, Laser, Missile, Shield, Heal
- Infinite levels
- No map selection
- Optimized boss transition to avoid freezing

## DSA Applied
- Queue: enemy spawning
- Stack: pause state
- ArrayList: bullets, enemies, eggs, items
- HashMap: asset loading
- Recursion: wave generation
